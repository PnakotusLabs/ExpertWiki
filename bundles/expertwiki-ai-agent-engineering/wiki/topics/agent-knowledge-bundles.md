# Agent Knowledge Bundles

## Summary

An agent knowledge bundle is a portable, source-preserving package that gives
an agent durable context without hiding the evidence. ExpertWiki applies the
local Markdown knowledge-wiki pattern to expert, project, topic, and viewpoint
records.

## Key Points

- Raw sources stay available under [Raw Sources](../../raw/sources/).
- Knowledge cards add structured metadata, quality state, freshness, and links.
- Agents can query the bundle locally before a hosted context service exists.
- The format is useful as a foundation for MCP, JSON graph, and `llms.txt`
  delivery layers.

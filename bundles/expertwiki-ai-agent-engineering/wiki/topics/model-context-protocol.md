# Model Context Protocol

## Summary

Model Context Protocol is an agent integration protocol for connecting AI
systems to external tools and data sources.

## Key Points

- MCP is relevant to LLM Wiki tools because agents need stable ways to read,
  search, and maintain local knowledge.
- A local CLI can come first, with an MCP adapter added after the file contract
  is stable.

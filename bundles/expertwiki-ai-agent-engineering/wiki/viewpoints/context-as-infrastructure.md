# Context As Infrastructure

## Claim

For expert and project knowledge, retrieval alone is not enough. A useful
context layer should preserve who said what, the evidence, the quality state,
and whether the information is still current.

## Supporting Evidence

- The local knowledge-wiki source supports a durable, compounding Markdown
  layer for agent-maintained context.
- File Search represents the retrieval layer that finds relevant material at
  query time.

## Applicability

This viewpoint is the strategic basis for a Context7-like distribution layer
and the future ExpertContext API. It is a synthesis, not a claim made by either
source.

# LLM Wiki And Retrieval

## Summary

Retrieval systems help find relevant source material at query time. An LLM Wiki
adds a maintained Markdown layer where agents can preserve synthesis, links,
open questions, and changes over time.

## Key Points

- Retrieval is useful for locating relevant documents.
- Wiki pages preserve synthesized context between sessions.
- The two approaches can work together when raw sources stay available.
